HTML Cheatsheet

Basic Tags

Name TAG Description Attributes
HTML <html> </html> Creates an HTML document
Header <head> </head> Sets off the title & other info that isn't displayed
Body <body> </body> Sets off the visible portion of the document
Title <title> </title> Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked

Text Tags

Name TAG Description Attributes
Preformatted Text <pre> </pre> Creates preformatted text
Headline <h1> </1>...<h6> </6> Creates headlines -- H1=largest, H6=smallest
Body <body> </body> Sets off the visible portion of the document
Bold Text <b> </b> Creates bold text (should use instead)
Italic Text <i> </i> Creates italicized text (should use instead)
Source Code <code> </code> Used to define source code, usually monospace
Citation <cite> </cite> Creates a citation, usually processed in italics
Address Section <address> </address> Creates address section, usually processed in italics
Emphasize <em> </em> Emphasizes a word (usually processed in italics)
Strong <strong> </strong> Emphasizes a word (usually processed in bold)
Font Size <font size=?> </font> Sets size of font - 1 to 7 (should use CSS instead)
Font Color <font color=?> </font> Sets font color (should use CSS instead)
Font Face <font face=?> </font face> Defines the font used (should use CSS instead)
